# GULP

WARNING: you need at least 14 version of node.

Simple boilerplate for markup based on Gulp

Do it in terminal:
```
$ npm i
$ gulp
$ ... 
$ profit!
```

To make buid, run:
```
$ gulp build
```

# WEBPACK

Frontend boilerplate, based on Webpack, incuding React, Typescript & latest ES features

```
$ npm i
$ npm run dev 
```
To make buid, run:
```
$ npm run build
```
